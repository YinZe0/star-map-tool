## 增加新副本支持
如需增加对新的副本、新的难度的支持，可在internal\strategy\strategies下增加策略类
然后在cmd\maptool\main.go中注册此策略


## 可能遇到的问题
- 程序和你抢键盘鼠标操控权
这是正常情况，因为程序是通过控制键盘鼠标信号实现的操作。
当程序运行时，尽量不要人为操控键盘、鼠标；
必要时，可以按F10，等待程序结束后自由操控键盘鼠标；

- 程序非F10方式停止后，游戏中的某个或部分键位仍在持续发挥作用
这是可能出现的，是由于非常规的方式停止程序，造成部分资源未能回收现象。
可以通过切换任意窗口解决


# 使用前提
1. 游戏任意分辨率窗口化
2. 设置 > 操控 > PC操作设置（镜头水平灵敏度、镜头垂直灵敏度 = 3）
3. 图像质量 = 流程
4. 敌方特效技能可见度 = 极简
